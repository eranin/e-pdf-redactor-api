Eranin PDF Redaction API

Mask or remove sensitive PDF regions by bounding boxes — FastAPI + PyMuPDF

API

POST /redact-pdf with a PDF file and request_data (JSON string).

curl -X POST http://localhost:8000/redact-pdf \
  -F "pdf_file=@/path/to/input.pdf" \
  -F 'request_data={{
    "pages_coordinates": {{
      "0": [{{"x1":50,"y1":50,"x2":200,"y2":100}}],
      "1": [{{"x1":100,"y1":120,"x2":240,"y2":180}}]
    }}
  }}'

Returns the redacted PDF.
